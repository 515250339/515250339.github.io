## 变量的介绍

1. 变量的概念 

   变量相当于内存中一个数据存储空间的表示，你可以把变量看做是一个房间的门 

   牌号，通过门牌号我们可以找到房间，同样的道理，通过变量名可以访问到变量 

   (值)。
