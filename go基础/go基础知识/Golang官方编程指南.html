就是我们 Go 的各个包的各个函数。

Golang 标准库 API 文档

1) API （Application Programming Interface,应用程序编程接口）是 Golang 提供的基本编程接口。

2) Go 语言提供了大量的标准库，因此 google 公司 也为这些标准库提供了相应的 API 文档，用于告

诉开发者如何使用这些标准库，以及标准库包含的方法。
